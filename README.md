# Calculator – my first real project

I built this calculator to learn how HTML, CSS, and JavaScript work together.  
It does basic math, handles decimals, clears with C, and shows "Error" if you divide by zero.

## How to run it
- Put `index.html`, `style.css`, and `script.js` in one folder.  
- Double‑click `index.html` – it works in any browser.

## How I made it
- The HTML and CSS are boilerplate – I let AI write those.  
- But the JavaScript logic (click handling, math, error checks) is **100% mine** – that's where I learned the most.

## What's next
Maybe keyboard support, a backspace button, or a square root.  
I'll write those myself too – because that's how you get better.
